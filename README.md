# CommunityConnect

CommunityConnect is a mobile-based community management platform designed for local regions, villages, wards, panchayats, and residential communities. It enables families, organizers, and local authorities to digitally manage invitations, announcements, attendance tracking, and community notifications in a centralized and organized manner.

## Repository Structure

This is a monorepo containing the following components:

- `mobile/`: The Flutter mobile application.
- `admin/`: (Planned) The React web application for the admin dashboard.

## Tech Stack

- **Mobile App**: Flutter
- **Backend / Services**: Firebase (Firestore, Authentication, Cloud Messaging)
- **Admin Panel**: React.js (Planned)
